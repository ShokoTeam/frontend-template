# Nuxt 3 Minimal Starter

#### Содержимое
- Nuxt 3
- Typescript 
- Eslint 
- Tailwind css
- Shadcn ui

### Что нужно для запуска:
- node v20.10.0
- npm v10.2.3
- pnpm (latest)


## Установка
```bash
pnpm install
```

## Запуск проекта

Переходи на `http://localhost:3000`:

```bash
pnpm run dev
```
